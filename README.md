# TetrisClone
Tetris Clone within Console
Developed on linux, may need some changes for it to work on Windows

Compile for release: gcc --std=gnu23 -Ofast *.c -o app -lm

Controls:
W - Hard drop
A - Move left
S - Soft drop
D - Move right
C - Hold piece
ESC - pause/unpause
, - Rotate counterclockwise
. - Rotate clockwise
